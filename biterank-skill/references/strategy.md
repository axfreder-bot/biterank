# BiteRank — Product & Strategy Quick Reference

## One-Line Pitch
> "BiteRank is the only app that ranks dishes, not restaurants — think Tinder for food discovery with the engagement of Duolingo."

## Two Value Propositions

### VP1: "Find the best restaurants for specific dishes"
- Dish-first search: "best tonkotsu ramen in Seattle"
- Community reviews specific to DISHES, not restaurants
- Rankings are dish-specific

### VP2: "A dating app for food"
- Food Swiper: swipe right to save, left to skip
- Passive discovery when bored
- Gamified XP/rewards create habit loops

- DAU/MAU ratio (target: 30%+)

- Return within 24h rate
- Review submission rate per swipe

## Monetization Priority

1. **DoorDash affiliate** — $3/user + $1,000/restaurant referral. Priority #1.
2. **Freemium** — $4.99/mo: unlimited reviews, advanced filters, no ads
3. **Sponsored discovery** — Restaurants pay to appear in "Near me" (clearly labeled)
4. **White label / B2B** — License dish-ranking data to restaurant chains

## Competitive Positioning

| Competitor | BiteRank Advantage |
|------------|-------------------|
| Yelp | Dish-first, not restaurant-first |
| Google Maps | Food-first curation |
| DoorDash / Uber Eats | Discovery-focused, not just ordering |
| Tinder for food clones | Real dish data + gamification |

## Gamification Core Loop
```
Swipe → Match → Add to list → Visit → Review → Earn XP → Level Up
```

Key habit loops:
- **Daily streak** — "Swipe 5 dishes a day" (Duolingo model)
- **Pioneer** — First to review a dish = bonus XP + badge
- **Dish completions** — Progress bar per restaurant
- **Social proof** — "X people added this dish this month"

## iOS App Strategy

**Phase 1 (NOW):** Wrap web app with Capacitor, ship to App Store
**Phase 2 (3 months):** Push notifications, haptic feedback, camera integration
**Phase 3 (6 months):** Apple Watch companion, social features

**App Store keywords:** food finder, dish reviews, restaurant rankings, food swiper

## Key Risks
1. **Cold start** — Seeded reviews already address this
2. **Discovery without intent** — Position as "bored? Just swipe"
3. **Yelp/Google dominance** — Don't compete on their turf; own "dish-first"
4. **Gamification without substance** — Lead with utility, support with gamification
